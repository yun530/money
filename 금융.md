# 1억 RPG — 게임형 저축 앱 기획안

> 금융 카테고리 | 웹 기반 싱글페이지 애플리케이션
> **컨셉: 저금할 때마다 레벨업, 1억 달성이 최종 보스**

---

## 1. 프로젝트 개요

| 항목 | 내용 |
|------|------|
| 프로젝트명 | 1억 RPG (One Hundred Million Quest) |
| 카테고리 | 금융 — 게임형 저축 달성 |
| 타깃 유저 | 저축에 동기부여가 필요한 20~30대 |
| 플랫폼 | 웹 (HTML/CSS/JS, 반응형) |
| 핵심 컨셉 | 저금 = 경험치 획득 / 1억 = 최종 보스 클리어 |
| 데이터 저장 | localStorage (회원별 분리, 비밀번호 SHA-256 해싱) |

### 한 줄 요약
> "저금할 때마다 레벨업하고, 칭호가 바뀌고, 1억에 가까워질수록 세상이 달라진다"

## 2. 핵심 게임 메커닉

### 2.1 레벨 시스템

총 10레벨. 누적 저금액 기준으로 자동 레벨업.

| 레벨 | 누적 저금액 | 칭호 | 설명 |
|------|------------|------|------|
| Lv. 1 | 0 ~ 99만원 | 🪙 동전 수집가 | 첫 발을 내딛은 당신 |
| Lv. 2 | 100만원 ~ | 💵 만원 부자 | 백만원 돌파! |
| Lv. 3 | 300만원 ~ | 🐷 저금통 마스터 | 저축 습관이 잡혔다 |
| Lv. 4 | 500만원 ~ | 💼 오백만원 클럽 | 진지해지기 시작 |
| Lv. 5 | 1000만원 ~ | 🌟 천만원 돌파자 | 이제 진짜다 |
| Lv. 6 | 2000만원 ~ | 🏦 이천만원 수호자 | 통장이 두꺼워졌다 |
| Lv. 7 | 3000만원 ~ | 👑 삼천만원 귀족 | 남들이 부러워한다 |
| Lv. 8 | 5000만원 ~ | 💎 오천만원 다이아 | 절반이다! |
| Lv. 9 | 7000만원 ~ | 🚀 억대 진입자 | 거의 다 왔다 |
| Lv. 10 | 1억 | 🏆 억만장자 | 최종 보스 클리어! |

### 2.2 경험치 (EXP) 시스템

저금 금액에 비례해 경험치 획득. 다음 레벨까지 EXP 바로 시각화.

```
획득 EXP = 저금 금액 / 1,000
예: 5만원 저금 → 50 EXP 획득
```

레벨업에 필요한 EXP는 해당 레벨 구간의 필요 저금액 / 1,000.

### 2.3 미션 시스템

저금 행동에 따라 미션 완료 → 보너스 EXP 지급.

#### 영구 미션 (1회 달성)

| 미션 | 조건 | 보너스 EXP |
|------|------|-----------|
| 첫 저금 | 첫 번째 저금 | +100 |
| 10만원 한 번에 | 단건 10만원 이상 | +150 |
| 레벨업 달성 | 레벨 오를 때마다 | +500 |
| 중간 보스 (5000만원) | Lv.8 달성 | +1000 |
| 최종 보스 (1억) | Lv.10 달성 | +9999 |

#### 월별 챌린지 (매월 초기화, 반복 가능)

매월 1일 새로운 챌린지 세트가 주어진다. 완료하면 EXP와 월별 전용 뱃지 획득.

| 챌린지 | 조건 | 보너스 EXP |
|--------|------|-----------|
| 연속 저금 3일 | 해당 월 내 3일 연속 | +200 |
| 이번 달 5회 저금 | 해당 월 내 5회 | +300 |
| 이번 달 목표 달성 | 월 저금 목표액 달성 | +500 |
| 불꽃 저금사 | 해당 월 내 7일 연속 | +800 |

> **월별 목표액**: 유저가 직접 설정 (기본값: 지난달 저금액). 설정 안 하면 챌린지 미표시.

#### 삭제 처리 규칙

저금 내역 삭제 시:
- **EXP는 차감하지 않는다** — 이미 획득한 경험치는 유지
- **미션 진행 카운트는 롤백한다** — 예: "이번 달 5회" 카운트 중 삭제 시 카운트 -1
- **이미 완료된 미션은 취소하지 않는다** — 달성 순간의 기록을 존중
- **레벨은 내려가지 않는다** — totalSaved가 줄어도 현재 레벨 유지

### 2.4 칭호 & 아이템 (뱃지)

레벨업, 미션 완료 시 수집형 뱃지 획득. 프로필에 전시.

#### 영구 뱃지

| 뱃지 | 획득 조건 |
|------|----------|
| 🌱 새싹 저축러 | 첫 저금 완료 |
| ⚡ 스피드 저금 | 하루 3회 이상 저금 |
| 💯 올인 | 단건 100만원 이상 |
| 🎯 중간 보스 격파 | 5000만원 달성 |
| 👑 억만장자 | 1억 달성 |

#### 월별 뱃지 (해당 월 챌린지 달성 시)

| 뱃지 | 획득 조건 |
|------|----------|
| 🔥 불꽃 저금사 (N월) | 해당 월 7일 연속 저금 |
| 🏅 월간 목표 달성 (N월) | 해당 월 목표액 달성 |

월별 뱃지는 `"불꽃 저금사 (2026-03)"` 형태로 월이 기록되어 컬렉션으로 쌓인다.

## 4. 화면 구성

```
[인증 화면]
  ├── 로그인 탭
  └── 회원가입 탭
        ↓
[메인 화면]
  ├── 헤더 (닉네임, 레벨, 로그아웃)
  │
  ├── 캐릭터 카드
  │     ├── 현재 레벨 + 칭호
  │     ├── EXP 바 (현재 / 다음 레벨까지)
  │     └── 획득 뱃지 미리보기
  │
  ├── 1억 달성 진행 바
  │     ├── 현재 저금액 / 1억
  │     └── 달성률 %
  │
  ├── [💰 저금하기] 버튼 (메인 CTA)
  │
  ├── 미션 섹션
  │     ├── 진행 중 미션 카드
  │     └── 완료된 미션 (흐리게)
  │
  └── 저금 내역 리스트
```

**모달 레이어**
- 저금 입력 모달
- 레벨업 축하 모달 (애니메이션)
- 미션 달성 모달

## 6. UI / UX 방향

| 항목 | 방향 |
|------|------|
| 테마 | 다크 + 골드 포인트 (RPG 느낌) |
| 폰트 | 한글 고딕 계열 + 숫자는 굵고 크게 |
| 색상 | 검정/네이비 배경, 골드/노랑 강조, 레벨별 색상 변화 |
| 레벨업 | 전체 화면 플래시 + 파티클 이펙트 |
| EXP 바 | 초록→파랑→보라→골드 레벨 구간별 색상 |
| 미션 완료 | 팝업 + 뱃지 획득 연출 |
| 반응형 | 모바일 우선 |

### 레벨별 UI 변화
| 레벨 구간 | 배경 분위기 | EXP 바 색상 |
|-----------|------------|------------|
| Lv.1~3 | 차갑고 어두운 느낌 | 회색 → 초록 |
| Lv.4~6 | 따뜻해지는 느낌 | 파랑 → 보라 |
| Lv.7~9 | 화려해지는 느낌 | 보라 → 금색 |
| Lv.10 | 황금빛 | 풀 골드 |
