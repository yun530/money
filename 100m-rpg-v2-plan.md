# 1억 RPG v2 — 억만장자 퀘스트 업그레이드 기획안

> 금융 카테고리 | 웹 기반 싱글페이지 애플리케이션  
> **v2 핵심 변경: 거지키우기 스타일 캐릭터 육성 + 월별 통계 + 예상 달성일**

---

## 1. 프로젝트 개요

| 항목 | 내용 |
|------|------|
| 프로젝트명 | 1억 RPG v2 (억만장자 퀘스트) |
| 카테고리 | 금융 — 게임형 저축 달성 |
| 타깃 유저 | 저축에 동기부여가 필요한 20~30대 |
| 플랫폼 | 웹 (HTML/CSS/JS, 반응형) |
| 핵심 컨셉 | 거지 → 억만장자로 성장하는 캐릭터 육성 RPG |
| 데이터 저장 | localStorage (회원별 분리) |

### 한 줄 요약
> "동전 한 닢으로 시작한 거지가 저금할수록 부자로 성장하고, 1억을 모으면 억만장자가 된다"

## 2. v1 → v2 핵심 변경 요약

| 항목 | v1 | v2 |
|------|----|----|
| 캐릭터 표현 | 이모지 + 칭호 텍스트 | 레벨별 캐릭터 이미지 변화 + 클릭 반응 |
| 성장 시각화 | EXP 바 | 캐릭터 외형 변화 + 배경 환경 변화 |
| 통계 | 없음 | 월별 저금 막대 차트 |
| 달성 예측 | 없음 | 현재 페이스 기반 예상 달성일 계산 |
| 게임 느낌 | 레벨업 모달 | 캐릭터 클릭 인터랙션 + 환경 변화 |

## 3. 핵심 변경 사항 상세

### 3.1 거지키우기 스타일 캐릭터 육성 (신규)

#### 컨셉
레벨이 오를수록 캐릭터의 외형과 주변 환경이 눈에 띄게 변한다. 저금액이 쌓일수록 "가난한 거지 → 부자 → 억만장자"로 시각적 성장을 체감할 수 있다.

#### 레벨별 캐릭터 & 환경 변화

| 레벨 | 누적 금액 | 캐릭터 상태 | 배경 환경 | 착용 아이템 |
|------|----------|------------|----------|------------|
| Lv.1 | 0 ~ 99만 | 🧍 누더기 거지 | 🌧️ 어두운 골목 | 구멍난 옷, 깡통 |
| Lv.2 | 100만 ~ | 🧍 평범한 시민 | 🏘️ 주택가 | 일반 옷 |
| Lv.3 | 300만 ~ | 🚶 직장인 | 🏢 도심 거리 | 넥타이, 서류가방 |
| Lv.4 | 500만 ~ | 💼 대리 | 🏙️ 오피스 빌딩 | 정장, 노트북 |
| Lv.5 | 1000만 ~ | 😎 과장 | 🌆 도시 야경 | 명품 시계 |
| Lv.6 | 2000만 ~ | 🕴️ 팀장 | 🏬 고급 빌딩 | 명품 수트 |
| Lv.7 | 3000만 ~ | 👑 임원 | 🌃 펜트하우스 | 롤렉스, 벤츠 |

- 캐릭터 클릭 → 랜덤 대사 출력 (레벨별 다른 대사)
- 저금 직후 클릭 → 기뻐하는 애니메이션
- 레벨업 시 캐릭터가 변신하는 트랜지션 효과

### 3.2 예상 달성일 계산 (신규)

#### 계산 방식

```
일평균 저금액 = 최근 30일 저금 합계 / 30
남은 금액 = 1억 - 현재 저금액
예상 달성까지 남은 일수 = 남은 금액 / 일평균 저금액
예상 달성일 = 오늘 + 남은 일수
```

> 전체 누적 평균이 아닌 최근 30일 기준으로 계산. 초반 데이터 왜곡 방지 및 최신 저축 패턴 반영.

#### 표시 방식

- 메인 화면 상단에 항상 표시
- "현재 페이스라면 **2028년 7월 예상**" 형태로 강조
- 저금 페이스별 메시지:
  - 빠름 (월 100만 이상): "🔥 이 속도라면 금방이에요!"
  - 보통 (월 30~100만): "💪 꾸준히 가고 있어요!"
  - 느림 (월 30만 미만): "⚡ 조금 더 속도를 올려볼까요?"
- 저금 기록이 없으면: "저금을 시작하면 예상일을 알 수 있어요"
- 최근 30일 저금이 없으면: "최근 한 달 저금 기록이 있으면 예상일을 계산할 수 있어요"

#### 다음 레벨 달성 예측 (추가)
- 예상 달성일 배너 하단에 **다음 레벨까지의 예측도 함께 표시**
- "Lv.6까지 80만원, 이 페이스면 **25일 뒤**" 형태
- 1억 달성이 멀게 느껴질 때 단기 목표로 동기부여 역할

### 3.3 월별 저금 통계 차트 (신규)

#### 표시 내용
- 최근 6개월 월별 저금액 막대 차트
- 이번 달 저금액 강조 (골드 컬러)
- 월 평균선 표시 (점선)
- 최고 달성 월 뱃지 표시

#### 차트 구성
```
[막대 차트 — 최근 6개월]
  10월  11월  12월  1월   2월   3월
  █     ██    ███   █     ████  ██
 30만  55만  80만  20만  110만 60만

평균: 59만원 / 월
최고: 2월 (110만원) 🏆
```

#### 추가 통계 수치
- 총 저금 횟수
- 가장 많이 저금한 요일
- 최대 단건 저금액
- 연속 저금 최장 기록 (일)

## 7. 예상 달성일 알고리즘 상세

```
1. 저금 기록이 없는 경우 → "저금을 시작하면 예상일을 알 수 있어요"

2. 최근 30일 저금이 없는 경우 → "최근 한 달 저금 기록이 있으면 예상일을 계산할 수 있어요"

3. 저금 기록이 있는 경우 (최근 30일 기준)
   - 최근 30일 저금 합계 계산 (오늘 기준 -30일 이내 트랜잭션 합산)
   - 일평균 저금액 = 최근 30일 합계 / 30
   - 남은 금액 = 1억 - 현재 총 저금액
   - 남은 일수 = 남은 금액 / 일평균 저금액
   - 예상 달성일 = 오늘 + 남은 일수

4. 다음 레벨 달성 예측
   - 다음 레벨 기준 금액 - 현재 총 저금액 = 다음 레벨까지 남은 금액
   - 다음 레벨까지 남은 일수 = 다음 레벨까지 남은 금액 / 일평균 저금액

5. 페이스 판단 (월 환산)
   - 월 100만원 이상 → 🔥 빠름
   - 월 30 ~ 100만원  → 💪 보통
   - 월 30만원 미만   → ⚡ 느림
```
